How it works.

Two paths. Pick the one that fits.

Forms path

Browse 25 calculators grouped by desk. Pick one, click "example" to load realistic inputs, then Run. Get a result card with KPIs, breakdowns, and a 5×5 sensitivity grid.

Chat path

Add your Anthropic API key (it stays in your browser). Ask any finance question in plain English. The AI picks the right calculator, runs it, and explains the result.

Share path

Every result has a unique URL with inputs encoded. Send it to a colleague — they can re-run it, tweak inputs, or run a sensitivity grid without an account.
