# Board Report

Prompt library for quarterly board presentations. Covers KPI dashboards, financial reviews, strategic discussions, and executive summaries.

## Role Context

```
You are a CFO preparing materials for a board of directors. Board members have
limited time and high expectations. Every slide must earn its place. Lead with
the headline, support with data, and end with the ask. Do not bury bad news —
surface it early with context and a plan.
```

---

## 1. Quarterly KPI Dashboard

```
Build a quarterly KPI dashboard for [company name] for [quarter].

Key metrics:
| KPI | Q Actual | Q Target | vs Target | Prior Q | YoY |
|-----|----------|----------|-----------|---------|-----|
| Revenue / ARR | $[X] | $[X] | | $[X] | |
| Gross Margin | [X]% | [X]% | | [X]% | |
| Net New ARR | $[X] | $[X] | | $[X] | |
| NRR | [X]% | [X]% | | [X]% | |
| CAC Payback | [X]mo | [X]mo | | [X]mo | |
| Burn Multiple | [X]x | [X]x | | [X]x | |
| Headcount | [X] | [X] | | [X] | |
| Runway | [X]mo | | | [X]mo | |

For the dashboard:
1. **Traffic light status** for each KPI (green/yellow/red)
2. **Trend arrows** (improving/stable/declining)
3. **Commentary**: 1-2 sentences per red/yellow KPI explaining why and what's being done
4. **Full-year outlook**: Are we on track for annual targets? If not, what changed?
```

## 2. Strategic Discussion Framing

```
Frame a strategic discussion for the board on [topic]:

Context: [describe the strategic question — e.g., "Should we move upmarket to enterprise?"]

Current state:
- [Key metric 1]: [value]
- [Key metric 2]: [value]
- [Key metric 3]: [value]

Structure the discussion:
1. **The question** (one sentence)
2. **Why now** — what triggered this discussion?
3. **Option A**: [describe] — Pros, cons, financial impact, timeline
4. **Option B**: [describe] — Pros, cons, financial impact, timeline
5. **Option C**: Status quo — what happens if we don't decide?
6. **Recommendation** with supporting data
7. **The ask**: What decision does the board need to make today?

Keep each option to 1-2 slides. Board members should be able to form a view in 5 minutes of reading.
```

## 3. Investor Update

```
Draft a quarterly investor update email for [company name].

Performance summary:
- ARR: $[X]M ([X]% YoY growth)
- Key wins: [1-3 notable customer wins or milestones]
- Key challenge: [1 honest challenge with plan]
- Cash position: $[X]M, [X] months runway
- Next milestone: [what you're focused on next quarter]

Guidelines:
1. **Lead with the headline** — one sentence on the quarter
2. **Numbers first**: 3-5 KPIs with context
3. **Wins**: Specific customer names, deal sizes, strategic significance
4. **Challenges**: Be direct — investors respect transparency over spin
5. **Outlook**: What are you optimizing for next quarter?
6. **Ask**: If you need anything from investors (intros, advice, bridge), say so

Tone: Confident, concise, data-driven. Not salesy. Under 500 words.
```
