# Investor Update & Earnings Prep

Prompt library for investor relations, earnings preparation, and stakeholder communication.

## Role Context

```
You are an investor relations professional who translates operating performance into
narratives that public and private investors can act on. You understand that markets
react to surprises — your job is to set expectations accurately, frame results in
context, and guide investors toward the metrics that matter most for this business.
```

---

## 1. Earnings Script Preparation

```
Prepare earnings talking points for [company name] for [quarter].

Results:
| Metric | Actual | Consensus | Beat/Miss | Guidance |
|--------|--------|-----------|-----------|----------|
| Revenue | $[X]M | $[X]M | | $[X-Y]M |
| EPS | $[X] | $[X] | | $[X-Y] |
| [Key metric] | [X] | [X] | | [X-Y] |
| [Key metric] | [X] | [X] | | [X-Y] |

Guidance update: [raising/maintaining/lowering] — new range: $[X-Y]M

Prepare:
1. **Opening headline**: One sentence framing the quarter
2. **3 key messages** to emphasize (what do you want investors to remember?)
3. **Beat/miss explanations**: For each line item vs. consensus
4. **Guidance bridge**: Walk from prior guidance to new guidance — what changed?
5. **Anticipated tough questions** and prepared responses (5-8 Q&A pairs)
6. **Competitive positioning statement**: How we compare to peers this quarter
```

## 2. Annual Shareholder Letter

```
Draft key sections of the annual shareholder letter for [company name].

Year in review:
- Revenue: $[X]M (vs. $[X]M prior year)
- Major milestones: [list 3-5]
- Key investments made: [list 2-3]
- Market position: [describe competitive standing]

Include:
1. **Opening narrative**: Where we started the year and where we ended
2. **Strategy update**: Progress against stated multi-year priorities
3. **Capital allocation**: How we deployed cash — R&D, M&A, buybacks, dividends
4. **Market context**: Industry tailwinds/headwinds and our positioning
5. **Forward look**: 2-3 priorities for the coming year
6. **Risk acknowledgment**: What could go wrong, and how we're prepared

Tone: Authentic, long-term oriented, Buffett-style clarity. No jargon.
```
